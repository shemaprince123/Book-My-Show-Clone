Building an application
1. npx create-react-app . 
2. npm start
3. npm install -D tailwindcss postcss autoprefixer
4. npx tailwindcss init
5. npm i babel (optional package in case tailwind changes are not loading well as expected  )
6.npm install react-slick (give the styles)
7. npm install slick-carousel 
8. npm i axios (mainly used to work with the API)
9. npm install @headlessui/react
10. npm install react-icons 
11. npm i react-router-dom
